[**Lihat Demo Live ↗️**](https://first-portofolio-one.vercel.app/)

---

## ✨ Fitur Utama
- **Modern UI/UX**: Menggunakan pendekatan minimalist dengan whitespace yang lega.
- **Fully Responsive**: Tampilan optimal di perangkat mobile, tablet, maupun desktop.
- **Tailwind CSS v4**: Menggunakan standar terbaru framework CSS untuk styling yang efisien.
- **Component-Based**: Struktur kode rapi dan modular (Refactored).
- **AOS Animation**: Animasi scroll yang halus untuk pengalaman interaktif.

## 🛠️ Tech Stack
- **Frontend:** React JS
- **Styling:** Tailwind CSS v4
- **Animation:** AOS (Animate On Scroll), Animate.css
- **Deployment:** Vercel

## 📁 Struktur Proyek
```text
